# Torque (UJIAN JWD)

- Author: Ronaldo Kereh
- Github: https://github.com/kereh

Project ujian sertifikasi JWD

# Installation

```sh
$ unzip ronaldo_kereh.zip
$ cd torque
$ npm install
$ npm db:push
$ npm run dev
```

# Tech Stack

- Next.js
- Tailwindcss (shadcn-ui)
- tRPC
- Drizzle ORM
- SQLite
